# C++ 영화 추천 및 통계 프로그램

숭실대학교 컴퓨터학부 C++ 프로그래밍 과제(M4) 영화 추천 시스템입니다.
평점 데이터를 바탕으로 사용자 간 유사도를 계산해서 영화를 추천해주는 프로그램입니다.

## 1. 개요
사용자가 영화에 매긴 평점을 사용해 피어슨 유사도를 계산하고, 가장 취향이 비슷한 유저가 높게 평가한 영화 중 내가 보지 않은 영화를 추천합니다. 
이번 M4 단계에서는 장르 필터링 추천, 장르별 평균 평점 그래프 출력, 상위 영화 랭킹 조회, 그리고 통계 결과를 파일로 내보내는 기능을 추가했습니다.

## 2. 빌드 및 실행 방법
Makefile이 작성되어 있어 make 명령어로 컴파일하고 실행할 수 있습니다.

```bash
# 빌드 방법
make clean && make

# 실행 방법
./movie_recommender
```

## 3. 프로그램 폴더 구조 및 소스 파일
- data/ : movies.csv, users.csv, ratings.csv 데이터 폴더
- include/ : 클래스 헤더 파일 (.h)
- src/ : 클래스 구현 파일 (.cpp)
- Makefile : 빌드 스크립트

## 4. 구현한 확장 기능 설명

### 장르 필터 추천
- 영화 추천 시 원하는 장르를 입력하면 해당 장르의 영화만 걸러서 추천 리스트를 보여줍니다.
- 장르를 입력하지 않고 엔터를 치면 기존처럼 모든 장르 영화를 추천합니다.

### 통계 및 시각화 기능
- 전체 영화의 평균 평점을 계산해 출력합니다.
- 장르별 평균 평점을 터미널창에 가로 막대 그래프(= 기호 조합)로 그려서 보여줍니다.
- 평점 상위 N개의 영화 목록을 부분 정렬(std::partial_sort)을 사용해 보여줍니다.
- 통계 분석 결과를 data/statistics_report.txt 파일로 저장하는 기능을 제공합니다.

### CLI 화면 개선
- 표(Table) 형태로 결과를 깔끔하게 보여줍니다. 한글 제목의 폭 정렬 알고리즘을 추가해서 세로줄이 깨지지 않고 잘 맞습니다.
- 추천 연산이나 통계를 계산할 때 진행률 표시바와 로딩 스피너 애니메이션을 제공합니다.
